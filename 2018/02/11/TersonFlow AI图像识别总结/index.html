TersonFlow AI图像识别总结

发表于 2018-02-11 | 分类于 AI |

最近在学习AI图像识别在Android 中的使用，看了非常多的资料，自己整理一下从训练模型到客户端使用训练好的模型进行图像识别的过程。

什么是AI

人工智能（Artificial Intelligence），是计算机科学的一个分支，它企图了解智能的实> 质，并生产出一种新的能以人类智能相似的方式做出反应的智能机器，该领域的研究包括机> 器人、语言识别、图像识别、自然语言处理和专家系统等。

AI的使用场景

从目前来看，我们可以利用AI进行语音识别、语音合成、文字识别、人脸识别、机器翻译、图像识别等的一些操作应用。比如，2018年央视纪录片《创新中国》，就是世界首部利用人工智能模拟人声完成配音的大型纪录片，节目解说部分全程运用人工智能配音，已逝“配音大师”李易的声音将在信息技术的支持下“重现”。

人工智能的实现——机器学习和神经网络（CNN）

机器学习是目前实现人工智能的最主要的方式，而神经网络是一种模拟人脑，取其精华取其精华去其糟粕的计算架构，利用神经网络进行机器学习，则让计算机不再只是执行命令的机器,具有了一定程度上举一反三的能力，这种能力就是人工智能。

AI深度学习移动端开源框架

以上总结了这么多概念性的东西，是为了方便我们理解AI和实现AI的方法。目前AI深度学习移动端的开源框架，主要有几种，百度MDL（Mobile-deep-learning），腾讯NCNN，还有Google的TersorFlow,在这里，我选取了TensorFlow作为技术模型框架。
TensorFlow是google开发的一个深度学习系统，我们可以通过Tensorflow Mobile API在移动端和嵌入式上部署模型，近几年来，由于其作为机器学习模型的使用已成倍增长，所以移动设备和嵌入式设备也出现了部署需求。Google又推出了更轻量级的TensorFlow Lite，针对移动设备进行了优化，使机器学习模型设备能够实现低延迟的推理。
TensorFlow Lite更是与Android 8.1中发布的神经网络API完美配合使用。目前国内很多家公司也都在使用TensorFlow,比如小米，京东，网易，英语流利说等。

为什么要在手机上运行TensorFlow?

一般AI识别的流程，是首先在服务器端训练模型=>客户端封装数据流请求=>服务器处理数据识别=>最后返回客户端处理结果，而通过网络连接发送设备访问的所有数据可能非常昂贵和耗时。在移动设备上运行使得App在不得不等待网络往返的情况下进行交互。所以需要把识别的工作放在本地操作，可以提高AI处理的速度和用户体验。

为什么叫TensorFlow？

Tensor（张量，代表N维数组），Flow（流，代表基于数据流图的是计算），把N维数字从流图的一端流动到另一端的过程，就是人工智能神经网络进行分析和处理的过程

Object Detection API组件

要想实现图像识别，我们可以借助TensorFlow其中一个工具，Object Detection API（对象探测API）组件，我们可以通过训练神经网络，借助这个对象探测API识别一个框架中的物体，Object Detection API中提供了5种网络结构的预训练的权重，全部基于coco图片训练集。
这里讲的coco（Common Objects in Context），其实是微软团队提供的一个可以用来进行图像识别的图片数据集。

TensorFlow安装训练环境搭建

关于训练环境的搭建这里就不多说了，网上可以找到很详细的步骤。我整理了一下大致的步骤和所需要安装的模块。

  • python安装和配置
  • 安装pillow（python的图片处理模块PIL），matplotlib（Python 的 2D绘图库），lxml（用来处理 XML 的第三方 Python 库）

  • 通过PIP命令安装tensorFlow 

    pip install tensorflow

  • 下载TensorFlow的源码
    编译源码需要NDK、Bazel工具

  • 下载TensorFlow模型源码
    https://github.com/tensorflow/models

  • 安装Google的 Protocol Buffer
    这是一种轻便高效的结构化数据存储格式

  • 解压模型源码
    在models/research/目录下执行protoc object_detection/protos/*.proto –python_out=.，编译Protobuf文件，生成py文件.

开始训练自己的数据集

收集样本图片

在开始训练自己的模型前，我们首先要收集样本图片，这里我准备通过机器学习训练模型识别人体器官——手的一个训练集。所以，我们可以通过一些插件或者工具收集各种各样关于手部的一个图片集。收集样本图片，可以使用chrome的插件Fatkun Batch Download Image，也可以使用ImageNet http://www.image-net.org/ ，这是目前世界是图像识别最大的数据库。

标记样本图片识别特征

在完成样本图片的收集后，我们需要在每张样本图片中标记出我们要识别的物体的特征范围，以便机器学习。同样，我们也可以借助工具完成，在windows上，我们可以使用labelImg标注工具，而在Mac OS上，我们可以使用RectLabel标注工具进行样本图片特征的标注，无论使用哪个工具，标注完图片特后，都会生成一份对应图片的xml格式的文件，这个xml就是图片识别特征的相关信息了，比如识别的范围，坐标，大小等。

数据格式转换

我们需要将标记完生成的xml文件转换为csv格式的文件，然后将csv格式的文件转换喂TensorFlow可以识别读取的TFRecord格式的文件。接着，我们就可以创建标签分类的配置文件label_map.pbtxt了。

使用预训练模型

深度学习成功的一个关键性因素就是大量带标签数据组成的训练集，我们实际训练的数据集过少，引入迁移学习fine-tuning（微调）的思想，对预训练模型进行微调，应用到自己的数据集上，形成自己的CNN（神经网络）。预训练的模型具有非常强大的泛化能力，这些预训练的模型的中间层包含非常多一般性的视觉元素，我们只需要对他的后几层进行微调，再应用到我们的数据上，通常就可以得到非常好的结果。

选择适当的预训练模型

我们可以在

  • https://github.com/tensorflow/models/blob/master/research/object_detection/g3doc/detection_model_zoo.md

上选择一个适当的预训练模型下载到电脑上。我们在这里选择了ssd_mobilenet_v1_coco作为我们的预训练模型，在这里，我们先看两个概念

  • SSD：single shot multibox detector，基于单一前馈神经网络的神经网络结构
  • MobileNet：卷积神经网络特征提取器，它们在手机上运行，计算消耗小、运行速度快，因此很适合在移动端上做应用

下载完预训练模型并解压后，我需要找到并修改ssd_mobilenet_v1_coco.config配置文件，我们可以修改num_classes，fine_tune_checkpoint，train_input_reader，eval_input_reader等这些配置要素。
